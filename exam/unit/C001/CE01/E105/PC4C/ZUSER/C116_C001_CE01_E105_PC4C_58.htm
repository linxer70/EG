다음의 관계를 참고하여 부서이름이 “영업부”인 사원들의 성명과 근무년수를 검색하는 SQL 구문으로 옳은 것은?
①  SELECT
성명, 근무년수 FROM 사원 WHERE 부서코드
   IN(SELECT 부서이름 FROM 부서
   WHERE 부서이름 = ‘영업부’);
②  SELECT
성명, 근무년수 FROM 사원 WHERE 부서코드
   IN(SELECT 부서코드 FROM 부서
   WHERE 부서이름 = ‘영업부’);
③  SELECT
A.성명, A.근무년수 FROM 사원 AS A, 부서 AS B
  WHERE B.부서이름 = ‘영업부’;
④  SELECT
성명, 성별 FROM 사원 WHERE 부서코드
   LIKE (SELECT 부서코드 FROM 부서
   WHERE 부서이름 = ‘영업부’);
