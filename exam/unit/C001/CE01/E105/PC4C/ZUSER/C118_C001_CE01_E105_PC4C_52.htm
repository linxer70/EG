어느 DB의 [제품] 테이브에는 ‘제품번호’와 ‘제품명’ 필드가 있고, [주문] 테이블에는 ‘주문번호’, ‘제품번호’, ‘주문수량’ 필드가 존재한다. 이때, ‘제품명’별로 ‘총주문수량’을 검색하는 SQL 문장으로 옳은 것은?
①  SELECT 제품.제품명, Sum(주문.주문수량) AS 총주문수량
  FROM 제품 inner join 주문
  on 주문.제품번호=제품.제품번호 GROUP BY 제품.제품명;
②  SELECT 제품.제품명, Sum(주문.주문수량) AS 총주문수량
  FROM 제품, 주문 GROUP BY 제품.제품명;
③  SELECT 제품.제품명, Sum(주문.주문수량) AS 총주문수량
 FROM 제품 inner join 주문
  on 주문.주문번호=제품.제품번호 GROUP BY 제품.제품명;
④  SELECT 제품.제품명, Sum(주문.주문수량) AS 총주문수량
  FROM 제품, 주문 GROUP BY 제품.제품명
  HAVING 주문.주문번호=제품.제품번호;
