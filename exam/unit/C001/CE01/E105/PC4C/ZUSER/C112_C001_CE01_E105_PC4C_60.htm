<창고>와 <제품>의 테이블 관계가 다음과 같을 때, 창고위치가 ‘부산'인 창고에 저장된 제품의 “제품명", “단가", “창고명"만을 표시하도록 하는 SQL 문은?
①  SELECT 제품.제품명, 제품.단가, 창고.창고명 FROM 창고 inner join 제품 WHERE 제품.창고위치 = "부산";
②  SELECT * FROM 창고 inner join 제품 on 창고.창고코드 = 제품.창고코드 WHERE 제품.창고위치 = "부산";
③  SELECT 제품.제품명, 제품.단가, 창고.창고명 FROM 제품 inner join 창고 on 창고.창고코드 = 제품.창고코드 WHERE 창고.창고위치 = "부산";
④  SELECT 제품.제품명, 제품.단가, 창고.창고명 FROM 창고 inner join 제품 HAVING 제품.창고위치 = "부산" AND 부서.부서번호 = 사원.부서번호;
