아래의 기본 테이블을 이용한 질의의 결과 레코드가 3개인 것은 무엇인가? (단 테이블에는 화면에 표시된 7개의 데이터만 들어있다.)
①  Select 부서, SUM(판매액) AS 판매합계
From 판매현황 Group By 부서;
②  Select 성별, AVG(판매액) AS 판매평균
From 판매현황 Group By 성별;
③  Select 부서, COUNT(부서) AS 사원수
From 판매현황 Group By 부서
Having COUNT(부서)>2;
④  Select 부서, COUNT(판매액) AS 사원수
From 판매현황 Where 판매액 >=1000000
Group By 부서;
