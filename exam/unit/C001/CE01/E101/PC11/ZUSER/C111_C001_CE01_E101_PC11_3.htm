프로그래밍 언어들에 대한 설명 중 올바르지 않은 설명은 어느 것인가?
①  ASP는 Active Server Page라고 하며 서버측 스크립트 언어로 마이크로소프트사(MS사)에서 제공한 웹언어이다.
②  PHP는 서버측 스크립트 언어로서 Linux, Unix, Windows 운영체제에서 사용가능하다.
③  JSP 스크립트는 JSP 페이지에서 자바를 삽입할 수 있으며, JSP 페이지에 실질적인 영향을 주는 프로그래밍을 할 수 있다.
④  XML 문서들은 SGML 문서 형식을 따르고 있으며 SGML은 XML의 부분집합이라고도 할 수 있기 때문에 응용판 또는 축약된 형식의 XML이라고 볼 수 있다.
