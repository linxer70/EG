한글 Windows XP운영체제 에서 OLE에 대한 설명으로 옳은 것은?
①  x86 계열 컴퓨터에서 시작 시에 하드웨어를 검사하고 운영 체제를 시작하며 하드웨어 장치 사이의 데이터 전송을 지원하는 필수 소프트웨어 루틴 집합이다.
②  어떤 응용 프로그램에서 만든 정보를 다른 응용 프로그램에서 만든 스프레드시트나 워드 프로세싱 파일 등의 문서에 붙여넣어 응용 프로그램간에 정보를 전송하고 공유하는 방법이다.
③  다양한 이동 컴퓨터, 데스크톱 컴퓨터, 서버 컴퓨터 및 주변 기기의 전원 관리를 정의하는 개방형 산업 규정이다.
④  드라이브 자체에 컨트롤러 장치가 내장되어 있어 별도의 어댑터 카드가 필요하지 않은 디스크 드라이브 인터페이스의 일종으로 예측 캐싱과 같은 향상된 기능을 제공 하여 전체적인 성능을 향상시킨다.
