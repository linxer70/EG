워크시트 출력시 머리글 또는 바닥글에 페이지 번호가 포함되어 있을 경우, 시작 페이지 번호를 100으로 지정하려고 한다. 다음 중 설명이 옳은 것은?
①  [페이지 설정]-[여백] 탭에서 [바닥글] 상자에 워크시트의 시작 페이지에 표시할 페이지 번호를 입력한다.
②  [페이지 설정]-[페이지] 탭에서 [자동 맞춤]에서 [용지 높이]를 워크시트의 시작 페이지에 표시할 페이지 번호를 입력한다.
③  [페이지 설정]-[페이지] 탭에서 [시작 페이지] 상자에 워크시의 시작 페이지에 표시할 페이지 번호를 입력한다.
④  [페이지 설정]-[설정] 탭에서 [시작 페이지] 상자에 자동이라고 입력한다.
