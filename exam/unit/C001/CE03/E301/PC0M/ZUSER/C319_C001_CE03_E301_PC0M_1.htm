다음은 한글 Windows XP에서 '잘라내기'해서 '붙여넣기'하는 작업 과정이다. (가)와 (나)의 빈칸에 해당하는 바로 가기 키로 옳은 것은?
- 이동할 파일을 선택한다.
- 그 파일을 클릭하고 ( (가) ) 를 누른다.
- 붙여 넣기 할 위치(폴더)로 이동한 후 ( (나) )를
  누른다.
①  (가) [Ctrl] + [C] (나) [Ctrl] + [V]
②  (가) [Ctrl] + [X] (나) [Ctrl] + [V]
③  (가) [Ctrl] + [C] (나) [Ctrl] + [X]
④  (가) [Ctrl] + [V] (나) [Ctrl] + [X]
