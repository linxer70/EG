인터넷 서비스에 관한 설명으로 옳지 않은 것은?
①  인터넷 도메인 이름에 해당하는 IP 주소와 DNS 정보를 검색하는데 이용되는 명령어로서 dig와 유사한 기능을 수행하는 명령어는 nslookup이다.
②  고퍼는 미국 미네소타 대학에서 개발된 검색 서비스로 데이터를 색인화하여 정보를 제공한다.
③  IRC 서버에 접속하여 대화를 나눌 때 사용하는 명령어 중 현재 개설된 채널, 사람 수, 제목 등을 보고자 할 때 사용하는 명령어는 /topic이다.
④  아키(Archie) 서비스는 Anonymous FTP 서버를 대상으로 사용자가 원하는 파일이 어느 위치에 있는지 알려주는 서비스이다.
