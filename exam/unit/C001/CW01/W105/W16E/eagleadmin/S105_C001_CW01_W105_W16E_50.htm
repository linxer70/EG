다음 중 웹 서비스에 관련된 설명으로 옳지 않은 것은?
①  하이퍼링크, 하이퍼텍스트, 하이퍼미디어 기술을 기반으로 하여 전 세계에 흩어져 있는 정보를 사용자가 원하는 대로 찾아볼 수 있도록 해주는 서비스이다.
②  웹 서버와 클라이언트가 상호 통신하기 위해 사용되는 웹 프로토콜을 HTTP(Hyper Text Transfer Protocol)이라 한다.
③  웹 페이지 작성에 사용되는 표준 언어는 Java Script이다.
④  웹의 주 기능은 멀티미디어를 포함한 모든 컴퓨터의 정보 처리기능, 통합 환경 제공, 쉬운 사용자 인터페이스이라 할 수 있다.
