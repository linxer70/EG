다음 설명의 ( ) 안에 들어갈 용어들을 순서대로 올바르게 나열한 것은?
마이크로프로세서(Microprocessor)의 처리능력은 보통 두 가지로 나타낸다. 하나는 데이터 처리능력을 나타내는 버스(Bus)의 ( ) 수이고, 다른 하나는 ( )당 발생하는 클럭(Clock)의 주파수인 ( )로 나타낸다.
①  비트(bit), 분(minute), 밉스(mips)
②  바이트(byte), 분(minute), 헤르츠(hertz)
③  비트(bit), 초(second), 헤르츠(hertz)
④  바이트(byte), 초(second), 밉스(mips)
