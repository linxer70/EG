다음 중 전환이 바르지 않은 문장을 고르시오.
①  I get so nervous that I can't breathe well.
→ I get too nervous to breathe well.
②  I'm so tired that I can't stand any longer.
→ I'm too tired to stand any longer.
③  The tree is so small that he can climb it.
→ The tree is small enough for him to climb.
④  The man is so heavy that he can't run fast.
→ The man is too heavy to run fast.
⑤  He runs so fast that no one beat him.
→ He runs too fast for anyone to beat him.
