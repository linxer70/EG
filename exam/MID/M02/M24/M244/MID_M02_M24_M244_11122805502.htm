혈액은 우리 몸을 순환하면서 생활에 필요한 영양분을 공급해 주고 영양분을 산화하여 생활에 필요한 생활에너지로 사용할 수 있도록 산소를 공급해 주는 역할을 할 뿐 아니라 노폐물과 이산화탄소를 받아서 몸밖으로 내보낼 수 있도록 물질을 운반해 주는 역할을 담당하고 있다.
다음 중 혈액 속의 산소를 조직으로 공급해 주고 이산화탄소를 받아오는 순환로를 바르게 나타낸 것은?
①  우심실 → 폐동맥 → 폐(모세혈관) → 폐정맥 → 좌심방
②  우심실 → 대동맥 → 온몸(모세혈관) → 대정맥 → 좌심방
③  좌심방 → 좌심실 → 폐(모세혈관) → 우심방 → 우심실
④  좌심실 → 폐동맥 → 폐(모세혈관) → 폐정맥 → 우심방
⑤  좌심실 → 대동맥 → 온몸(모세혈관) → 대정맥 → 우심방
